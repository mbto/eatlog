* Keep a log of consumed products
* See a graph of your weight change
* Track weight, consumed products and excess calorie consumption of other users

* **With tests:**

* **Without tests:**
    * Unix: `gradlew build -x test`
    * Windows: `gradlew.bat build -x test`
